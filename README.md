# 花丸札

生活タスク用のGitHub Pages向けPWA。

普通のタスク管理ではなく、頭の中の札を剥がして、終わった札を花丸スタンプに変換するための小さい道具。

## MVP機能

- PWA化
- ローカル保存
- カテゴリ切替
- 単発／繰り返し登録
- 期限あり／いつでも
- いつでも札の表示数設定
- 今日のおすすめハイライト
- 完了タップ
- 花丸スタンプ変換
- 単発削除
- 繰り返しリセット
- 二重カウント防止
- 提出文生成
- Undo
- JSONエクスポート／インポート
- AI向け依頼文生成とタグ形式取り込み

## GitHub Pages

Repository settings → Pages → Deploy from a branch → `main` / `/root` を選ぶ。
